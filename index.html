Projects

Bloom is a desktop application designed for plant enthusiasts, sellers, and buyers. Built with JavaFX, it features a dual-interface system for easy plant sales and care management. Sellers can upload plant details and care instructions, while customers can browse plants and manage their collections. Integrated care tools like watering reminders and care guides help simplify plant care. Data is efficiently stored and managed using MySQL.

IoT-based Laser Tag game offers an interactive experience with smart vests equipped with laser sensors, RGB LEDs, and ultrasonic sensors. Powered byESP32 microcontrollers, the vests process real-time hit data, which is transmitted to the cloud via ThingSpeak and stored in Firebase. Players can view live scores and leaderboards on the website.

Haseel is an interactive educational game designed to teach students about sustainable green living and space exploration. It uses GLOBE protocols to simulate real-world scientific methods and data collection. The game helps develop critical thinking and problem-solving skills by comparing players' data with GLOBE standards. Targeted at school students aged 6-16, Haseel aims to provide educational value through engaging gameplay
